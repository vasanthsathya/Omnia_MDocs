# Overview

Omnia is an open-source deployment toolkit by Dell Technologies that automates the setup of HPC, AI, and data analytics clusters on Dell PowerEdge servers. This section explains Omnia's architecture, components, and design concepts.

If you are new to Omnia, start here to understand what it does and how it works before proceeding to [Get Started](../GetStarted/index.md).
